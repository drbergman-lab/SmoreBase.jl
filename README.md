SmoreBase.jl is a Julia package that defines the core functionality for the SMoRe framework, which is designed for surrogate modeling of complex systems. This package provides the foundational tools and structures necessary for building surrogate models, including data handling, model fitting, and evaluation metrics. It serves as the base layer upon which more specific surrogate modeling techniques can be developed and implemented in other packages within the SMoRe ecosystem.
